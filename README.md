# Mini Social Post Application

A full-stack social feed web application using React.js (with Basic CSS), Node.js, Express, and MongoDB.

## Project Structure

- `frontend/`: Contains the React.js client application, created with Vite.
- `backend/`: Contains the Node.js and Express server with Mongoose for MongoDB.

## Local Setup Instructions

### 1. Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. *(Optional)* Setup your MongoDB URI in a `.env` file if required by the implementation.
4. Start the backend developement server (You can add a start script in package.json):
   ```bash
   node index.js
   ```

### 2. Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
